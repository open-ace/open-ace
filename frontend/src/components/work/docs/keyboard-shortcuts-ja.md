# キーボードショートカットガイド

## 概要

Open ACE は以下のキーボードショートカットをサポートし、ユーザーがシステムをより効率的に使用できるようにします。

> ヒント：macOSユーザーはCmdキーを、Windows/LinuxユーザーはCtrlキーを使用します。

## ワークスペース/タブ管理

- **Cmd+Shift+<** (macOS) / **Ctrl+Shift+<** (Win/Linux): 前のタブに切り替え
- **Cmd+Shift+>** (macOS) / **Ctrl+Shift+>** (Win/Linux): 次のタブに切り替え

## AI会話インターフェース（iframe内）

- **Cmd+左矢印** / **Ctrl+左矢印**: 前のタブに切り替え
- **Cmd+右矢印** / **Ctrl+右矢印**: 次のタブに切り替え

## フルスクリーンモード

- **Esc**: ワークスペースフルスクリーンモードを終了
- **Esc**: 会話履歴フルスクリーンモードを終了

## ポップアップ/モーダル

- **Esc**: 現在のポップアップ/モーダルを閉じる
- **Enter**: フォームを送信または操作を確認

## ドロップダウンメニュー

- **Esc**: ドロップダウンメニューを閉じる

## 検索可能選択ボックス

- **Esc**: 選択ボックスを閉じて検索内容をクリア
- **Enter**: 最初に一致するオプションを選択

## 会話の名前変更

- **Enter**: 新しい会話名を保存
- **Esc**: 名前変更操作をキャンセル

## 入力ボックス送信

- クォータ管理: **Enter** クォータ設定を保存
- クォータアラート: **Enter** アラート設定を保存

## 確認ダイアログ

- **Enter**: 確認操作を実行

---

上記のショートカットはすべて現在のバージョンで実装されています。
