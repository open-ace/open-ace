# 키보드 단축키 가이드

## 개요

Open ACE는 다음 키보드 단축키를 지원하여 사용자가 시스템을 보다 효율적으로 사용할 수 있도록 합니다.

> 팁: macOS 사용자는 Cmd 키를, Windows/Linux 사용자는 Ctrl 키를 사용합니다.

## 워크스페이스/탭 관리

- **Cmd+Shift+<** (macOS) / **Ctrl+Shift+<** (Win/Linux): 이전 탭으로 전환
- **Cmd+Shift+>** (macOS) / **Ctrl+Shift+>** (Win/Linux): 다음 탭으로 전환

## AI 대화 인터페이스 (iframe 내)

- **Cmd+왼쪽 화살표** / **Ctrl+왼쪽 화살표**: 이전 탭으로 전환
- **Cmd+오른쪽 화살표** / **Ctrl+오른쪽 화살표**: 다음 탭으로 전환

## 전체 화면 모드

- **Esc**: 워크스페이스 전체 화면 모드 종료
- **Esc**: 대화 기록 전체 화면 모드 종료

## 팝업/모달

- **Esc**: 현재 팝업/모달 닫기
- **Enter**: 양식 제출 또는 작업 확인

## 드롭다운 메뉴

- **Esc**: 드롭다운 메뉴 닫기

## 검색 가능 선택 상자

- **Esc**: 선택 상자 닫기 및 검색 내용 지우기
- **Enter**: 첫 번째 일치 옵션 선택

## 대화 이름 변경

- **Enter**: 새 대화 이름 저장
- **Esc**: 이름 변경 작업 취소

## 입력 상자 제출

- 할당량 관리: **Enter** 할당량 설정 저장
- 할당량 알림: **Enter** 알림 설정 저장

## 확인 대화상자

- **Enter**: 확인 작업 실행

---

위의 모든 단축키는 현재 버전에서 구현된 기능입니다.
