# 键盘快捷键使用指南

## 概述

Open ACE 支持以下键盘快捷键，帮助用户更高效地使用系统。

> 提示：macOS 用户使用 Cmd 键，Windows/Linux 用户使用 Ctrl 键。

## 工作区/标签页管理

- **Cmd+Shift+<** (macOS) / **Ctrl+Shift+<** (Win/Linux): 切换到上一个标签页
- **Cmd+Shift+>** (macOS) / **Ctrl+Shift+>** (Win/Linux): 切换到下一个标签页

## AI 对话界面(iframe 内)

- **Cmd+左箭头** / **Ctrl+左箭头**: 切换到上一个标签页
- **Cmd+右箭头** / **Ctrl+右箭头**: 切换到下一个标签页

## 全屏模式

- **Esc**: 退出工作区全屏模式
- **Esc**: 退出会话历史全屏模式

## 弹窗/模态框

- **Esc**: 关闭当前弹窗/模态框
- **Enter**: 提交表单或确认操作

## 下拉菜单

- **Esc**: 关闭下拉菜单

## 可搜索选择框

- **Esc**: 关闭选择框并清空搜索内容
- **Enter**: 选择第一个匹配选项

## 重命名会话

- **Enter**: 保存新的会话名称
- **Esc**: 取消重命名操作

## 输入框提交

- 配额管理: **Enter** 保存配额设置
- 配额告警: **Enter** 保存告警配置

## 确认弹窗

- **Enter**: 执行确认操作

---

以上快捷键均为当前版本已实现的功能。
