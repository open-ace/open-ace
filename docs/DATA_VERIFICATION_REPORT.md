## 🔍 数据问题分析

### 问题 1: 今日用量为空

**现象**: Dashboard 的"今日用量"区块显示"暂无数据"

**原因**: 
- 当前日期：2026-03-21
- 最新数据日期：2026-03-20
- 数据收集脚本尚未运行或今日确实无使用

**建议**:
1. 运行数据收集脚本：`python3 scripts/fetch_qwen.py` 等
2. 或者等待今日实际使用后数据自动收集

**状态**: ✅ 正常行为，非 Bug

---

### 问题 2: Messages 列表为空

**现象**: Messages 页面显示 0 条消息

**原因**:
- 消息表 (`messages_YYYY-MM-DD`) 中无数据
- 消息数据可能需要单独收集

**建议**:
1. 检查消息收集脚本是否运行
2. 检查日志文件路径配置
3. 手动导入历史消息数据

**状态**: ⚠️ 需要数据收集

---

### 问题 3: Token 分布图为空

**现象**: Dashboard 右侧的 Token 分布饼图显示"暂无数据"

**原因**:
- 分布图依赖 `todayData`
- 今日用量为空导致分布图也为空

**建议**:
- 与问题 1 相同，需要收集今日数据

**状态**: ✅ 正常行为，非 Bug
